The project contains the following compiler directives:
1. REMOVE_OCR_PLUGIN - allows to disable code, which references to VintaSoft OCR .NET Plug-in
2. REMOVE_OCR_ML_ASSEMBLY - allows to disable code, which references to VintaSoft OCR .NET Plug-in, Vintasoft.Imaging.Ocr.ML assembly
3. REMOVE_PDF_PLUGIN - allows to disable code, which references to VintaSoft PDF .NET Plug-in
4. REMOVE_DOCCLEANUP_PLUGIN - allows to disable code, which references to VintaSoft Document Cleanup .NET Plug-in
5. REMOVE_JBIG2_PLUGIN - allows to disable code, which references to VintaSoft JBIG2 .NET Plug-in
6. REMOVE_JPEG2000_PLUGIN - allows to disable code, which references to VintaSoft JPEG2000 .NET Plug-in
7. REMOVE_BARCODE_SDK -  allows to disable code, which references to VintaSoft Barcode SDK
